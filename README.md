# customer_behaviour_analysis

# Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow, starting from data loading and exploration to generating insights and building an interactive dashboard. The objective is to extract meaningful patterns from raw data and present them in a clear, actionable format for decision-making.

---

## Dataset

* The dataset used in this project contains structured data relevant to the analysis objective.
* It includes multiple features such as categorical and numerical variables.
* Data quality issues such as missing values, duplicates, and inconsistencies were addressed during preprocessing.

---

## Tools & Technologies

* **Python** – Data loading, cleaning, and exploratory data analysis (EDA)
* **SQL (PostgreSQL / MySQL / SQL Server)** – Data querying and transformation
* **Power BI** – Dashboard creation and visualization
* **Libraries** – Pandas, NumPy, Matplotlib, Seaborn

---

## Project Steps

### 1. Data Loading

* Imported dataset into Python using Pandas
* Performed initial inspection of structure and data types

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions and relationships between variables
* Identified trends, patterns, and anomalies
* Visualized key insights using charts and graphs

### 3. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized and formatted data for consistency

### 4. SQL Analysis

* Loaded cleaned data into relational databases
* Wrote SQL queries to:

  * Aggregate data
  * Perform joins and filtering
  * Extract business insights

### 5. Dashboard Development

* Built an interactive dashboard in Power BI
* Included KPIs, charts, and filters for better usability
* Enabled dynamic data exploration

### 6. Reporting

* Summarized key findings and insights
* Highlighted trends and actionable recommendations

---

## Dashboard

* Interactive Power BI dashboard showcasing:

  * Key performance indicators (KPIs)
  * Trend analysis
  * Category-wise breakdowns
* Designed for easy interpretation by non-technical stakeholders

## Results & Insights

* Identified key trends and patterns in the dataset

## How to Run

1. Clone the repository:

   ```bash
   git clone <your-repo-link>
   ```

2. Install required Python libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```

3. Run the Python scripts/notebooks for:

   * Data loading
   * EDA
   * Data cleaning

4. Import the cleaned dataset into your SQL database (PostgreSQL/MySQL/SQL Server)

5. Execute SQL queries provided in the `/sql` folder

6. Open the Power BI file (`.pbix`) to view the dashboard
